###Installation
1. Navigate to the root directory of the project
2. Install all node dependencies
```
npm install
```
3. Install Bower for client dependencies
```
npm install -g bower
```
4. Install client dependencies
```
bower install
```

###Running Site
1. Once all dependencies have been installed above
```
node server.js
```
