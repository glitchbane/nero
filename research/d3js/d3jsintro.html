A Quick Intro To d3js

  • d3js
  • General purpose library for manipulating documents based on data
    • Not a charting library, but you can make charts with it
    • Makes the connection between data and graphics easy
  • JavaScript, HTML, SVG, Canvas, CSS

  • Easily bind small/large data sets (JSON, CSV, data format of your choice) to visuals to produce something meaningful
  • Create custom, rich, interactive and expressive graphs with drilldowns

Pros

  • Open-source library (BSD license) with active development
  • Extensible (frameworks built on top of D3.js)
  • Build a custom visual experience for your data
  • Generating visuals client-side (scalable solution for large data sets) or server-side (reduce page load, better performance)
  • Can handle a lot of crazy math stuff
    • projections, treemaps, voronoi, quadtree, hull

Cons

  • Requires a very good understanding of JavaScript
  • Manipulation of every single element in the visuals
    • DOM manipulation is expensive for large numbers of elements
  • Doesn't have pre-built charts but there are libraries and wrappers:
    • c3js
    • nvd3
